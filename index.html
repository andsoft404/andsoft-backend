Танилцуулга

АндСофт Глобал Партнэр компани нь оюутан наснаас эхэлсэн нөхөрлөл, хамтын мөрөөдлөөс төрсөн мэдээлэл технологийн компани юм. Манай компани вебсайт болон мобайл аппликейшн хөгжүүлэлтийн чиглэлээр үйл ажиллагаа явуулдаг бөгөөд салбартаа хөгжиж буй чадварлаг залуу хамт олонтой.

Манай хамт олон 5 хүний бүрэлдэхүүнтэй бөгөөд програм хангамжийн хөгжүүлэлт, UI/UX дизайн, системийн дизайн зэрэг чиглэлүүдээр мэргэшсэн. Бид байгууллагын хэрэгцээ шаардлагад нийцсэн, орчин үеийн технологид суурилсан, найдвартай шийдлүүдийг санал болгон ажилладаг.

Үйл ажиллагааны чиглэл

  • Байгууллагын вебсайт хөгжүүлэлт

    Орчин үеийн технологид суурилсан, хариуцлагатай вебсайт хөгжүүлэлт.

  • Мобайл аппликейшн (iOS, Android)

    iOS болон Android платформд зориулсан мэргэжлийн аппликейшн хөгжүүлэлт.

  • Байгууллагын дотоод систем (CRM, ERP)

    Байгууллагын үйл ажиллагааг автоматжуулах дотоод системийн шийдэл.

  • Backend систем болон API хөгжүүлэлт

    Найдвартай backend систем, API интеграци болон UI/UX дизайны шийдэл.

Эрхэм зорилго ба Алсын хараа

Эрхэм зорилго

Орчин үеийн технологийн шийдлээр дамжуулан байгууллагуудын үйл ажиллагааг хялбарчилж, үр ашигтай болгоход хувь нэмэр оруулах.

Алсын хараа

Мэдээлэл технологид суурилсан, олон салбарыг хамарсан дэлхийн түншлэлийн экосистемийг бүтээх.

Хамт олон

  • Гүйцэтгэх захирал (CEO)

    Санхүү, маркетинг, компанийн ерөнхий удирдлагыг хариуцна.

  • Дэд захирал / CTO

    Системийн архитектур, Backend, Data analysis, нийт хөгжүүлэлт хариуцна. Developer баг удирдана.

  • Product Designer / UI UX

    Апп, вэбийн дизайн, хэрэглэгчийн туршлага (UX), интерфейс (UI) хариуцна.

  • Business Development Manager

    Харилцагчтай уулзалт, захиалга авах, маркетинг, сурталчилгаа хариуцна.

  • Frontend Architect

    Вэб, аппын хэрэглэгчид харагдах интерфэйс хөгжүүлэх. UI-г код болгох (Next, React, Flutter гэх мэт).
